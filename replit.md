# Static Website

## Overview
A static HTML/CSS website served via a simple Python HTTP server.

## Project Structure
- `home.html` - Main homepage
- `download.html` - Downloads page
- `style.css` - Main stylesheet
- `download/` - Download page assets and CSS
- `assets/img/` - Image assets
- `file/` - Downloadable files
- `server.py` - Python HTTP server for serving static files

## Running the Project
The server runs on port 5000 using Python's built-in HTTP server:
```
python server.py
```
